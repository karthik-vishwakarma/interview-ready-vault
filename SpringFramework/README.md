# Spring & Spring Boot Interview Topics

This section covers the Spring ecosystem, from core Dependency Injection to Spring Boot's auto-configuration and
Microservices integration.

## 🔍 Sub-Topics

### 1. [Spring Core & IoC](./Core/README.md)

* **Concepts:** Bean Lifecycle, ApplicationContext vs BeanFactory, Dependency Injection (DI), and Inversion of Control (
  IoC).
* **Focus:** Understanding the container that manages your application.

### 2. [Spring Boot Essentials](./SpringBoot/README.md)

* **Concepts:** Auto-configuration, Starters, `@SpringBootApplication`, and the Actuator.
* **Focus:** How Spring Boot simplifies configuration and production-readiness.

### 3. [Spring MVC & Web](./Web/README.md)

* **Concepts:** DispatcherServlet, RestController vs Controller, Content Negotiation, and Exception Handlers.
* **Focus:** Building RESTful APIs and web applications.

### 4. [Spring Data & Persistence](./Data/README.md)

* **Concepts:** Spring Data JPA, Repository abstraction, Transaction Management (`@Transactional`), and Hibernate
  integration.
* **Focus:** Efficient database communication and consistency.

### 5. [Spring Security](./Security/README.md)

* **Concepts:** Authentication vs Authorization, JWT Integration, OAuth2, and Filter Chains.
* **Focus:** Protecting your application and managing user identity.

### 6. [Testing & DevOps](./Testing/README.md)

* **Concepts:** `@MockBean`, `@SpringBootTest`, Testcontainers, and Profiles.
* **Focus:** Ensuring code quality and environment-specific configurations.
