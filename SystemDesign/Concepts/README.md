# System Design Interview Questions

This section contains essential system design questions categorized by difficulty. Mastery of these topics is crucial
for architectural rounds at top tech companies.

## 🚥 Difficulty Levels

* 🟢 [Easy: Fundamentals & Components](Easy.md)

*Focus: Load balancing, Caching, CAP Theorem, SQL vs NoSQL, and basic definitions.*

* 🟡 [Medium: Functional Service Design](Medium.md)

*Focus: Designing specific services like URL shorteners, Notification systems, and Rate limiters.*

* 🔴 [Hard: Large-Scale Distributed Systems](Hard.md)

*Focus: Multi-region availability, Real-time collaboration, Fraud detection, and High-concurrency scaling.*
