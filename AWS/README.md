# AWS Interview Topics

This section covers the Amazon Web Services (AWS) ecosystem, focusing on the Cloud Adoption Framework and the Well-Architected Framework.

## 🔍 Sub-Topics

### 1. [Compute (EC2, Lambda, ECS)](./Compute/README.md)
* **Concepts:** Instance types, Serverless (Lambda), Auto Scaling, and Containerization (Fargate).
* **Focus:** Choosing the right compute power for the workload.

### 2. [Storage (S3, EBS, EFS)](./Storage/README.md)
* **Concepts:** Object storage vs. Block storage, S3 Storage Classes, and Lifecycle policies.
* **Focus:** Cost-optimization and data durability.

### 3. [Networking (VPC, Route53, CloudFront)](./Networking/README.md)
* **Concepts:** Subnets, Security Groups vs. NACLs, VPC Peering, and Global Content Delivery.
* **Focus:** Security, isolation, and low-latency delivery.

### 4. [Databases (RDS, DynamoDB, Aurora)](./Databases/README.md)
* **Concepts:** Relational vs. NoSQL in the cloud, Multi-AZ vs. Read Replicas, and Provisioned vs. On-demand throughput.
* **Focus:** Scalability and high availability of data.

### 5. [IAM & Security](./Security/README.md)
* **Concepts:** Policies, Roles, Groups, Least Privilege principle, and AWS STS.
* **Focus:** Identity management and infrastructure security.
